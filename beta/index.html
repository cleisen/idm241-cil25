IDM 241 BETA DESCRIPTION

National Geographic Site


How It Works

Triggers

  • Hover on image of TV show on "Browse" page
  • Hover on Information button
  • Hover off Information button
  • Hover off image of TV show on "Browse" page

Rules

  • Hover on image:
    • Pointer changes to a finger
    • Circle outline with “i” fades in over time on left corner of image

  • Hover on Information button:
    • Translucent gray rectangle fades from 0 to 60% opacity over time
    • Rectangle rises from bottom to overlay the image

  • Hover off Information button
    • Translucent gray rectangle fades from 60% to 0 opacity over time
    • Rectangle moves down and disappears over time

  • Hover off image:
    • Circle outline with “i” on left corner of image fades out over time
    • Finger changes back to pointer
